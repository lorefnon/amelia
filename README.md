Amelia
======

Minimal bare-bones console RSS reader powered by Node.js

Installation
------------

[sudo] npm install -g amelia

 - Adding a feed:
   amelia --add techcrunch http://techcrunch.com/rssfeeds

 - List all feeds:
   amelia --list

 - Read a feed:
   amelia --read techcrunch

 - Removing a feed:
   amelia --purge techcrunch
